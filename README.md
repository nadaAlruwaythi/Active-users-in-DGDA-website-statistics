📊 نبذة عن البيانات (About Dataset)

تمثل هذه البيانات عدد المستخدمين النشطين لموقع هيئة تطوير بوابة الدرعية (DGDA) على أساس أسبوعي. حيث تعكس مستوى التفاعل مع الموقع ومدى استخدامه من قبل الزوار خلال فترات زمنية مختلفة.

تُستخدم هذه البيانات لتحليل اتجاهات الاستخدام، وفهم سلوك المستخدمين، وتقييم أداء الموقع الرقمي بمرور الوقت، مما يساهم في دعم اتخاذ القرارات المتعلقة بتحسين تجربة المستخدم وزيادة كفاءة الخدمات المقدمة.
